5분 이내에 Paperclip을 로컬에서 실행하세요.

## 빠른 시작 (권장)

```bash
npx paperclipai onboard --yes
```

설정을 안내하고, 환경을 구성하고, Paperclip을 실행합니다.

## 로컬 개발

**사전 요구사항:** Node.js 20+, pnpm 9+

```bash
pnpm install
pnpm dev
```

API 서버와 UI가 `http://localhost:3100`에서 시작됩니다. 외부 데이터베이스 불필요 — Paperclip은 기본적으로 임베디드 PostgreSQL을 사용합니다.

## 원커맨드 부트스트랩

```bash
pnpm paperclipai run
```

설정이 없으면 자동 온보딩하고, 헬스체크를 자동 복구하며, 서버를 시작합니다.
